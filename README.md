# Docker container images with "headless" VNC session

Each Docker image is installed with the following components:

* Desktop environment [**Xfce4**](http://www.xfce.org)
* VNC-Server (default VNC port `5901`)
* Browsers:
  * Mozilla Firefox
  * Google Chorme
